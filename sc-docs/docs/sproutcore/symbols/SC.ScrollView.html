Class SC.ScrollView

Extends: SC.View
Defined in: frameworks/sproutcore/frameworks/desktop/views/scroll.js
Availability: since SproutCore 1.0

Overview

Implements a complete scroll view. This class uses a manual implementation of scrollers in order to properly support clipping frames.

Important Events:

  • contentView frame size changes (to autoshow/hide scrollbar - adjust scrollbar size)
  • horizontalScrollOffset change
  • verticalScrollOffsetChanges
  • scroll wheel events

-   (SC.StackedView) updateHeight (immediately)

Parameters

(Boolean)immediately YES to update immedately

Returns

(SC.StackedView) receiver

Discussion

This is called automatically whenever an item view is reloaded. You can also call this method directly if the height of one of your views has changed.
