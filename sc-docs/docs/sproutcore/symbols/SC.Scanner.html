Class SC.Scanner

Extends: SC.Object
Defined in: frameworks/sproutcore/frameworks/foundation/system/datetime.js
Availability: since SproutCore 1.0
Author: Martin Ottenwaelter

Overview

A Scanner reads a string and interprets the characters into numbers. You assign the scanner's string on initialization and the scanner progresses through the characters of that string from beginning to end as you request items.

Scanners are used by DateTime to convert strings into DateTime objects.

Property Details

scanLocation [R/W]

The current scan location.

-   Object scanLocation

Discussion

It is incremented by the scanner as the characters are processed. The default is 0: the beginning of the string.

string [R/W]

The string to scan.

-   Object string

Discussion

You usually pass it to the create method:

SC.Scanner.create({string: 'May, 8th'});
