Class SC.WebView

Extends: SC.Control, SC.View
Defined in: frameworks/sproutcore/frameworks/desktop/views/web.js
Availability: since SproutCore 1.0

Overview

Used to display an iframe. The source, (specified by the value property) of the iFrame should be from the same domain. (i.e. the src / value should be from the same domain) if you want to access the contents of the iframe.

The content of the iframe can be bigger than the size specifed when creating the view.

-   Object shouldAutoResize

Discussion

If you want the view to be auto-resized to the dimensions of the iframe, then set the value of this property to YES. The web view can be auto resized only if the contents are from the same domain as the parent domain.
