Class SC.LabelView

Extends: SC.Control, SC.InlineEditorDelegate, SC.View
Defined in: frameworks/sproutcore/frameworks/foundation/views/label.js
Availability: since SproutCore 1.0

Overview

Displays a static string of text.

You use a label view anytime you need to display a static string of text or to display text that may need to be edited using only an inline control.

escapeHTML [R/W]

If true, value will be escaped to avoid scripting attacks.

-   Object escapeHTML

Discussion

This is a default value that can be overridden by the settings on the owner view.

The exampleInlineTextFieldView property is by default a SC.InlineTextFieldView but it can be set to a customized inline text field view.

Discussion

-   Object fontWeight

Discussion

Set this to a validator or to a function and the value will be passed through it before being set.

-   Object formatter

Discussion

This is a default default that can be overidden by the settings in the owner view.

hint [R/W]

The hint to display if no value is set.

-   Object hint

Discussion

Should be used only if isEditable is set to YES.

icon [R/W]

An optional icon to display to the left of the label.

-   Object icon
