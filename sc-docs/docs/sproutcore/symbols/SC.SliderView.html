Class SC.SliderView

Extends: SC.Control, SC.View
Defined in: frameworks/sproutcore/frameworks/desktop/views/slider.js
Availability: since SproutCore 1.0

Overview

A SliderView shows a horizontal slider control that you can use to set variable values.

You can use a slider view much like you would any other control. Simply set the value or content/contentValueKey to whatever value you want to display. You can also set the maximumValue and minValue properties to determine the mapping of the control to its children.

Property Details

contentMaximumKey [R/W]

Optionally specify the key used to extract the maximum progress value from the content object.

-   String contentMaximumKey

Discussion

If this is set to null then the maximum value will not be derived from the content object.

contentMinimumKey [R/W]

Optionally specify the key used to extract the minimum progress value from the content object.

-   String contentMinimumKey

Discussion

If this is set to null then the minimum value will not be derived from the content object.

handleSelector [R/W]

The DOM element that displays the handle.

-   Object handleSelector

Discussion

This element will have its top-left corner updated to reflect the current state of the slider. Use margin-offsets to properly position your handle over this location.

maximum [R/W]

The maximum value of the progress bar.

-   Object maximum

minimum [R/W]

The minimum value of the progress.

-   Object minimum

step [R/W]

Optionally set to the minimum step size allowed.

-   Object step

Discussion

All values will be rounded to this step size when displayed.

value [R/W]

Bind this to the current value of the progress bar.

-   Object value

Discussion

Note that by default an empty value will disable the progress bar and a multiple value too make it indeterminate.
